# Devin Integration Protocol

## ガイドライン読み込みプロトコル

1. タスク受信時の動作
   - `/docs/guidelines/README.md` を確認
   - プロジェクト固有のガイドラインの有無を確認
   - 優先順位に基づいて適用するルールを決定

2. 確認応答
   - ガイドライン確認後、規定のフォーマットで応答
   - 最新の更新日時を含める

3. 優先順位の処理
   - 優先度1-2のルールが存在する場合、それらを最優先
   - 共通ルール（優先度3）は、より高い優先度のルールがない限り適用
   - 優先度4-10のルールは補足的なガイドラインとして扱う
