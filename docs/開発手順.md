# AWS サーバーレスアプリケーション開発手順

## 概要
このドキュメントでは、AWSを使用したサーバーレスアプリケーションの開発手順について説明します。
アプリケーションは、React + TypeScript + Viteを使用したフロントエンドと、AWSのサーバーレスサービスを組み合わせて構築されています。

## 開発環境
- Node.js
- TypeScript
- Vite
- React
- Tailwind CSS
- shadcn/ui

## アプリケーションの機能
- 食事と飲み物の記録
- カスタムメニューの追加と削除
- 摂取量（mg）の記録（50mg単位、50-800mg）
- 日次合計の表示（食事、飲み物、総計）
- モバイルフレンドリーなUI

## 開発手順

### 1. フロントエンド開発
1. プロジェクトの初期化
   - Devin環境でReactアプリケーションを作成
   - TypeScript、Tailwind CSS、shadcn/uiを含む開発環境のセットアップ

2. 主要機能の実装
   - モード選択（食事/飲み物）の実装
   - メニュー項目の表示と選択機能
   - カスタムメニューの追加/削除機能
   - 摂取量選択機能（50mg単位）

3. データ管理の実装
   - LocalStorageを使用したデータ永続化
   - カスタムメニューのCSVエクスポート機能
   - 日次データの集計機能

4. UIの改善
   - タッチフレンドリーなボタンサイズ（最小44x44px）
   - レスポンシブデザインの実装
   - エラーメッセージの表示
   - スクロール可能なログ表示領域

### 2. デプロイ
1. アプリケーションのビルド
   - 本番用ビルドの生成
   - TypeScriptの型チェック

2. デプロイ
   - Devin環境を使用したデプロイ
   - フロントエンドのホスティング設定

## アクセス方法

https://aws-simple-app-pwf5ev4m.devinapps.com

## 注意事項
- データはブラウザのLocalStorageに保存されます
- カスタムメニューはCSVファイルとしてエクスポート可能です
- モバイルデバイスでの使用を推奨します
- 重複するメニュー項目は追加できません
